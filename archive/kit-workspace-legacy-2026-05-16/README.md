# Workspace Control Folder

This folder contains the folder-based workflow system.

| Path | Purpose |
|---|---|
| `map/` | Routing, naming, safety, and workspace map |
| `workflows/` | Task workflows and review checklists |
| `runbooks/` | Known commands and repeatable procedures |
| `context/` | Durable policy, style, architecture, and workflow context |
| `outputs/` | Staged plans, validations, reviews, and handoffs |
| `templates/` | Reusable artifact templates for plans, reviews, decisions, handoffs, and retrospectives |

Start with `AGENTS.md`, then `map/routing-table.md`, then only the workflow/runbook/context files required for the active task.
