# Naming Conventions

Use durable artifact names when saving staged outputs.

Pattern:

```text
YYYY-MM-DD-topic-stage-vNN.md
```

Common stages:

- `brief`
- `plan`
- `validation`
- `review`
- `handoff`

Examples:

```text
2026-05-15-opencode-folder-migration-plan-v01.md
2026-05-15-config-generation-validation-v01.md
2026-05-15-code-review-review-v01.md
```
