# Workspace Map

This repository is an OpenCode folder-workflow baseline.

## Folders

| Path | Purpose |
|---|---|
| `AGENTS.md` | Always-on operating policy and safety gates |
| `CLAUDE.md` | Claude Code pointer to `AGENTS.md` |
| `_workspace/map/` | Navigation, routing, naming, and safety rules |
| `_workspace/workflows/` | Task workflows and review checklists |
| `_workspace/runbooks/` | Known commands and repeatable procedures |
| `_workspace/context/` | Durable context and style guidance |
| `_workspace/outputs/` | Staged artifacts: plans, validations, reviews, handoffs |
| `_workspace/templates/` | Artifact templates |
| `.opencode/` | Runtime config, active prompt, disabled reference prompts, skills |
| `templates/` | Bootstrap templates for minimal, work, and personal workspaces |
| `scripts/` | Workspace initialization and verification scripts |
| `global/opencode/` | Global OpenCode instruction files |
| `archive/` | Completed or retired material |

## Context rule

Do not read the whole repository by default. Start with `AGENTS.md`, then `_workspace/map/routing-table.md`, then only the workflow/runbook/context files required for the active task.
