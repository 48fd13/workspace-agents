# Managed Service Integration Checklist

For cloud managed services, check:

- auth/identity model
- network access and private/public endpoint choice
- data ownership, retention, backup, and deletion behavior
- quotas, limits, throttling, and retry semantics
- local/dev/test strategy
- observability and error handling
- cost drivers
- vendor lock-in or portability constraints
- fallback, rollback, or migration path
