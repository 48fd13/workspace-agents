# Skill Import Checklist

Before importing a skill:

- [ ] Source repository and license are acceptable
- [ ] `SKILL.md` has clear `name` and `description`
- [ ] Trigger conditions are specific
- [ ] Instructions are actionable, not generic advice
- [ ] External tools/API calls are explicit
- [ ] Scripts are reviewed
- [ ] Secrets handling is safe
- [ ] Destructive/publishing actions require confirmation
- [ ] Skill belongs in this repo, not `personal-agents/`
- [ ] OpenCode restart is noted after install
