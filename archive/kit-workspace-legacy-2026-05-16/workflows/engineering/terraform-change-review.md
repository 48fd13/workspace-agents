# Terraform / IaC Change Review

Use when reviewing Terraform, OpenTofu, or similar IaC files and plans.

Use `_workspace/runbooks/terraform-plan-reading.md`.

Check creates/updates/deletes/replacements, state/backend impact, provider/module versions, variable defaults, naming/tagging, IAM/network/security changes, cost impact, and apply safety.

Never apply infrastructure changes without explicit approval.
