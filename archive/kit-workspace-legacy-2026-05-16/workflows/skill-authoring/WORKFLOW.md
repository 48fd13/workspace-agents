# Skill Authoring Workflow

Use to create or improve a reusable skill.

## Process

1. Confirm the repeated task the skill should cover.
2. Define when to use and when not to use it.
3. Keep frontmatter name and description precise.
4. Put workflow instructions in `SKILL.md`.
5. Put long examples, scripts, or references in subfolders only when needed.
6. Add safety gates for external actions, secrets, destructive work, or publishing.
7. Review with `_workspace/workflows/skill-review/WORKFLOW.md`.

Use `_workspace/runbooks/skill-import-checklist.md` for structure.
