# Folder Workflow Kit Docs

This repository is the reusable OpenCode/Claude folder-workflow kit.

It is not the daily machine-level workspace. The daily starting point is:

```text
/Users/spider/workspace
```

This repo owns:

- OpenCode runtime config and global setup
- reusable templates
- initialization and verification scripts
- reusable skills
- guidance for maintaining the kit

## Key docs

- `concepts.md`: root workspace vs kit vs target workspace
- `template-selection.md`: when to use `minimal`, `development`, or `personal`
- `kit-maintenance.md`: how to maintain this repository
- `mwp-conventions.md`: authoring rules for pipelines
- `mwp-compatibility.md`: how this kit maps to MWP concepts
- `workflow-kit-bootstrap.md`: cross-tool procedure for creating or repairing the kit
- `artifacts/`: canonical artifact skeletons copied into workspace templates
