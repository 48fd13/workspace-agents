# Naming Conventions

Use durable artifact names:

```text
YYYY-MM-DD-topic-stage-vNN.md
```

Common artifact locations:

```text
_workspace/outputs/handoffs/YYYY-MM-DD-topic-handoff-vNN.md
_workspace/outputs/reviews/YYYY-MM-DD-topic-review-vNN.md
_workspace/outputs/learning-material/YYYY-MM-DD-topic-learning-material-vNN.md
_workspace/outputs/source-notes/YYYY-MM-DD-topic-source-note-vNN.md
_workspace/outputs/processed-notes/YYYY-MM-DD-topic-processed-note-vNN.md
```
